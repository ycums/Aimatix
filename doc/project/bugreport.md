## Good
- **Good**: プレビューと確定された時刻が一致している ✅
- **Good**: 確定されている時刻が正しい ✅

## Bad
- **Bad**: 入力欄表示が期待と異なる
- 前提: 現在時刻 `00:00`
- raw input:  AAAAAB
- expected 
  - 入力欄表示: `__:50`
- actual
  - 入力欄表示: `__:5_`

## 解説
未入力の上位桁は `_` を維持して、下位の桁は初期値が `0` になる。

## 修正方針

B で 桁送り するときに 下記のようにする
```cpp
        digits[3] = 0;
        entered[3] = true; // ← ここを変更
```
